# Astrbot机器人主动传话插件

核心功能说明

1. 指令设计
核心指令：/relay + 目标会话标识 + 消息内容，支持群聊和私聊传话
帮助指令：/relay_help（别名：传话帮助、relay 用法），提供详细使用指引
会话标识格式：平台名称:消息类型:会话ID，严格遵循 AstrBot 统一会话标识规范

2. 主动消息发送
依赖 context.send_message() 实现主动推送，兼容所有支持主动消息的平台
自动拼接发送者昵称，让接收方明确消息来源
包含完整的异常捕获，处理会话不存在、平台不支持等场景

3. 平台适配支持
支持的平台：QQ 个人号 (aiocqhttp)、Telegram、飞书等（参考文档中 “主动消息” 支持矩阵）
不支持平台：QQ 官方接口、企业微信、钉钉等（会提示 “平台不支持主动消息”）

4. 使用注意事项
目标会话标识必须准确，格式错误会导致传话失败
主动消息功能依赖平台支持，使用前需确认目标平台是否在 “主动消息” 支持列表中
群聊会话 ID 为群号，私聊会话 ID 为对方用户 ID（如 QQ 号）
若出现 “未找到目标会话”，请检查平台名称、消息类型、会话 ID 是否均正确
